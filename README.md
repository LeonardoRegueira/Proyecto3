# SGT - Gestor de Tareas

## Descripcion

SGT es un proyecto integrador correspondiente al Grupo N.º 3.

El objetivo es crear una herramienta para organizar y administrar tareas de forma simple, ayudando a los usuarios a llevar un mejor control de sus actividades diarias.

## Tecnologias usadas

### Frontend

- Angular 20.3.x
- TypeScript 5.9.x
- CSS

### Backend

- Node.js
- JavaScript
- Express 5.2.x
- MongoDB
- Mongoose 9.6.x

## Que instalar antes de empezar

Para que el proyecto funcione, cada integrante debe tener instalado:

| Software | Version recomendada | Para que sirve |
| --- | --- | --- |
| Node.js | 20.19 o superior | Ejecutar backend y frontend |
| MongoDB Community Server | 7.x u 8.x | Base de datos del proyecto |

Para verificar si Node.js y npm estan instalados:

```bash
node -v
npm -v
```

## Instalar dependencias

Como la carpeta `node_modules` no se sube al repositorio, cada persona debe instalar las dependencias en su computadora.

### Backend

```bash
cd backend
npm install
```

### Frontend

```bash
cd frontend
npm install
```

## Comandos basicos para ejecutar

Para usar el proyecto completo, abrir dos terminales.

### Terminal 1: backend

```bash
cd backend
npm run dev
```

El backend queda funcionando en:

```text
http://localhost:3000/
```

Si todo esta bien, deberia mostrar:

```text
Backend funcionando
```

### Terminal 2: frontend

```bash
cd frontend
npm start
```

El frontend queda funcionando en:

```text
http://localhost:4200/
```
